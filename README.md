# ⚡ MotionForge

**A free, open-source Roblox Studio plugin for building custom combat moves visually.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Features

- 🎮 **Visual Timeline Editor** - Add hitboxes, sounds, and effects at specific times
- 👁️ **Hitbox Visualizer** - See your hitbox in real-time with color-coded damage
- 🎬 **Cutscene System** - Camera shake, zoom, and slow-motion effects and rotation
- 📦 **One-Click Export** - Generate ready-to-use Lua code
- 💾 **Save/Load** - Persistent move storage across Studio sessions
- 📚 **Template Library** - Built-in templates + save your own

## Installation

1. Download `MotionForge.rbxmx` from Releases
2. In Roblox Studio, go to **Plugins** → **Plugins Folder**
3. Place the file in the folder
4. Restart Studio

Or install directly from the **Creator Store** (search "MotionForge")

## How to Use

1. Open MotionForge from the Plugins tab
2. Set your move's name, damage, and cooldown
3. Click **"+ Add Event"** to add hitboxes at specific times
4. Use **"👁️ Show Hitbox Preview"** to visualize your hitbox
5. Click **"📦 Export"** to copy Lua code or export to ReplicatedStorage
